# Email Optimizer

> 10-point email audit: subject lines, hook, mobile formatting, PS, CTA, skimmability. Paste any email, get a better one back.

## Trigger Keywords
- "Optimize this email"
- "Review this email"
- "Polish this email before I send it"
- "Audit this email"
- "Make this email better"

## Process

Run the email draft through all 10 criteria in sequence, applying fixes as you go:

### 1. Subject Lines
Generate 10 subject line variants. Score each for curiosity, specificity, and open-rate potential. Select the strongest one and explain why.

### 2. Hook Check
Evaluate the first line. Does it earn the read? If not, rewrite it. The first sentence should create enough curiosity or value to keep reading — no throat-clearing, no "Hey [name], hope you're well."

### 3. Preamble Rewrite
Kill filler before the value starts. If the first 2-3 sentences don't deliver value or build tension, cut or rewrite them.

### 4. Mobile Formatting
Reformat for mobile reading:
- Short paragraphs (1-3 sentences max)
- White space between sections
- Scannable layout
- No walls of text

### 5. Read-Aloud Test
Read the email out loud (mentally). Flag and fix any awkward flow, clunky transitions, or sentences that don't sound natural when spoken.

### 6. PS Audit
Evaluate the PS line. If there isn't one, add one. A good PS reinforces the CTA, adds a secondary hook, or creates urgency. If the existing PS is weak, strengthen it.

### 7. CTA Alignment
Verify the call-to-action matches the email's content and promise. One clear CTA per email. If there are multiple competing CTAs, consolidate.

### 8. Preview Text
Write optimized preview text (the snippet shown in inbox before opening). Should complement the subject line, not repeat it.

### 9. Skimmability
Add bold text, bullets, and whitespace for fast readers. Most people skim first, read second — the email should deliver value to both.

### 10. Final Pass
Re-read the complete optimized email. Catch anything the individual steps missed. Ensure consistency in tone, formatting, and message.

## Output Format

Deliver two things:

### 1. The Optimized Email
The fully rewritten email — ready to send. Include:
- Winning subject line
- Preview text
- Rewritten body
- PS line

### 2. Audit Summary
A one-line summary for each of the 10 criteria showing what was changed and why.

```
| # | Criteria | Change |
|---|----------|--------|
| 1 | Subject Lines | Generated 10 variants, selected "..." for [reason] |
| 2 | Hook | Rewrote opening — original was [issue] |
| ... | ... | ... |
```
